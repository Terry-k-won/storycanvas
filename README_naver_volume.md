# 네이버 키워드 블로그/카페 월간 발행량 조회기

네이버 검색 API를 사용해 원하는 키워드의 **블로그 월간 발행량**과
**카페/블로그 누적 검색량**을 조회하는 파이썬 CLI 도구입니다.
외부 패키지 설치 없이 표준 라이브러리만으로 동작합니다.

## 1. 필요한 API 키

[네이버 개발자센터](https://developers.naver.com) → **Application → 애플리케이션 등록**
→ 사용 API에서 **검색**을 체크하면 아래 두 값이 발급됩니다.

| 항목 | HTTP 헤더 | 환경변수 |
|------|-----------|----------|
| 클라이언트 아이디 | `X-Naver-Client-Id` | `NAVER_CLIENT_ID` |
| 클라이언트 시크릿 | `X-Naver-Client-Secret` | `NAVER_CLIENT_SECRET` |

> 403 오류가 나면 애플리케이션의 **API 설정** 탭에서 '검색'이 선택돼 있는지 확인하세요.

## 2. 키 입력 방법

방법 A — 환경변수:
```bash
export NAVER_CLIENT_ID=발급받은_아이디
export NAVER_CLIENT_SECRET=발급받은_시크릿
```

방법 B — `.env` 파일 (`.env.example` 복사):
```bash
cp .env.example .env
# .env 파일을 열어 실제 값으로 수정
```

## 3. 사용법

```bash
# 최근 30일 기준
python naver_volume.py "강남 맛집"

# 특정 월 기준 (2026년 5월)
python naver_volume.py "강남 맛집" --month 2026-05

# 여러 키워드 일괄 조회 + CSV 저장
python naver_volume.py --keywords-file keywords.txt --csv result.csv
```

`keywords.txt` 예시 (한 줄에 하나씩):
```
강남 맛집
제주도 여행
홈트레이닝
```

## 4. 출력 예시

```
====================================================
키워드        : 강남 맛집
집계 기간     : 2026-05-07 ~ 2026-06-06
----------------------------------------------------
블로그 월간 발행량 : 842 건
블로그 누적(total) : 1,234,567 건
카페 누적(total)   : 89,012 건  *발행일 미제공으로 월간 집계 불가
====================================================
```

## 5. 동작 원리 & 한계

- **블로그**: `search/blog` API는 게시일(`postdate`)을 제공합니다.
  최신순(`sort=date`)으로 페이지를 훑으며 대상 기간에 실제 발행된 글 수를 셉니다.
- **API 한계**: 네이버 검색 API는 `start` 최댓값이 1000이라 최대 1000건까지만
  조회됩니다. 한 달 발행량이 1000건을 넘으면 `1000+ (한도초과)`로 표시됩니다.
- **카페**: `search/cafearticle` API는 발행일을 제공하지 않아 월간 발행량을
  정확히 셀 수 없습니다. 참고용으로 누적 검색결과 수(total)만 표시합니다.
